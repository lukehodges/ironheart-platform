4. Risk Radar

High Risk

Payment gateway not integrated — Stripe provider pattern exists but no live integration; blocks revenue flow.
Notification has zero tests — No test coverage, no frontend UI, multiple TODOs in handlers. Critical path for user experience.

Medium Risk

Jobs / Booking duplicate — Two overlapping modules cause confusion; needs merge before further work.
Calendar sync untested — Zero tests for sync logic; webhook edge cases may surface in production.
RBAC untested — Permission system has no automated tests; security-critical path.

Low Risk

Forms needs builder UI — Backend complete; drag-and-drop builder is additive, not blocking.
Review needs aggregation — Core flow works; averages and display widget are polish items.
Analytics needs optimization — Dashboard renders; pre-calculation and exports are performance enhancements.
